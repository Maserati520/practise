# Kigali Travel Website

Welcome to the Kigali Travel Website project! This website is a multi-page platform dedicated to showcasing some of the best tourist destinations, restaurants, and experiences in Kigali, Rwanda.

## 🌍 Overview

This project provides a beautiful, user-friendly interface for visitors to explore various attractions in Kigali. It features a modern design with rich aesthetics, dynamic animations, and responsive layouts.

## 🗺️ Destinations Featured

The website highlights several key locations:

- **Kigali Genocide Memorial** (`memorial.html`): A place of remembrance and learning.
- **Fazenda Sengha** (`fazenda.html`): An outdoor recreational center offering zip-lining, horseback riding, and more on Mount Kigali.
- **Heaven Restaurant** (`heaven.html`): A renowned culinary destination in Kigali.
- **Fusion Restaurant** (`fusion.html`): A great spot for diverse and delicious dining.

## 🛠️ Technologies Used

- **HTML5**: For semantic structuring of the web pages.
- **CSS3**: For custom styling, animations, responsive design, and layout.

## 🚀 Getting Started

To view the website locally:

1. Clone or download the repository to your local machine.
2. Open the `index.html` file in your preferred web browser.
3. Navigate through the website using the built-in links to explore different destinations.
